## Finding Tariff IDs

### Method 1: From OpenEI URL

When viewing a tariff on OpenEI, the ID is in the URL:


### Method 2: From Tariff Page

On the tariff details page, look for:
- "Rate ID" field
- "EIA ID" or similar identifier
- Copy the alphanumeric string (usually 24 characters)

## Tips for Users

1. **Bookmark Favorite Tariff IDs**: Keep a text file with commonly used tariff IDs
2. **Use Environment Variable**: Set `OPENEI_API_KEY` once instead of entering repeatedly
3. **Check Duplicates**: The app automatically handles duplicate names by adding numbers
4. **Verify Import**: Always check the success message to confirm correct tariff imported
5. **Refresh Page**: Remember to refresh after import to see new tariff in dropdown

## Common Issues and Solutions

| Issue | Solution |
|-------|----------|
| Button stays disabled | Verify both API key and tariff ID are entered |
| "Invalid API key" error | Double-check API key has no extra spaces |
| "Tariff not found" error | Verify tariff ID is correct and complete |
| Import seems slow | Normal - tariff data can be large, wait for spinner to finish |
| New tariff not in dropdown | Refresh the page or reselect the dropdown |
| File already exists | App automatically adds number suffix (file_1.json, file_2.json) |

## Keyboard Shortcuts

- **Tab**: Navigate between API key and tariff ID fields
- **Enter** (in tariff ID field): Submits form (if fields are filled)
- **Ctrl+V / Cmd+V**: Paste tariff ID

## Mobile/Responsive Behavior

The import section is fully responsive:
- On narrow screens: Fields stack vertically
- On wide screens: More space for input fields
- Button: Always full-width for easy tapping/clicking
